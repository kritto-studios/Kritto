# Kritto-cards

Kritto Eid Cards 🌙

Kritto is a creative project for sending beautiful digital Eid cards online.

This project creates interactive Eid greeting cards that open like a real card and allow people to send personalized messages.

✨ Features
- Interactive web-based Eid card
- Personalized "To" and "From" names
- Minimal and elegant design
- Easy to share with a link

Example:
?to=Sara&from=Aisha

Created by Kritto Studios.
